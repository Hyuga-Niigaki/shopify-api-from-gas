# なんだこれは：
大量のメタフィールドを手で作ると腱鞘炎になるから、実行ボタンを押したら一括で作成してくれるプログラム。
GASで動作します。

# 概要：
B列の値にて、商品メタフィールドか、バリアンとメタフィールドかを判別

C列をメタフィールド定義の名前とする

D列をメタフィールド定義のネームスペースとキーとする

G列をメタフィールド定義のタイプとする

H列をメタフィールド定義のタイプとする
